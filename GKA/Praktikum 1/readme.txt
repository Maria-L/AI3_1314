Instalation der Praktikums Aufgabe

1. Package graph einbinden

Ein leerer Graph kann nun mit
--	Graph graph = new Graph();
erstellt werden

2. Package graphReader einbinden. 
Sie müssen sicherstellen, dass die einzulesenden Graph-Dateien an der richtigen Stelle gespeichert sind. 
Der Ort zum Einlesen kann im Dokument "main.java" in Zeile 17 geändert werden.
Wenn die Packages graph und graphReader nicht im selben Projekt liegen, muss graphReader:main.java graph importieren.
